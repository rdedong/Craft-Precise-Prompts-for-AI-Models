# Craft-Precise-Prompts-for-AI-Models
Write prompts to generate specific outputs from LLMs

*Introduction*
In this lab, you will write and refine prompts to generate relevant output from large language 
models (LLMs).

**Software requirements**
To complete this lab, you don’t have to install any software on your computer. You only require 
a Google account to access Google Colab and a Replicate account to access and use various 
artificial intelligence (AI) models. 
Objective  
After completing this lab, you should be able to:  
• Write prompts to generate specific outputs from LLMs
